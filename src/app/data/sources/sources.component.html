Gather, clean, and recontextualize data

Raw data used in this project was gathered from a variety of sources, including OddsPortal and MoneyPuck, via web scraping and accessible CSV files. Data was cleaned and recontextualized to fit the needs of the project. This included parsing data into readable columns, converting data types, and merging tables.
